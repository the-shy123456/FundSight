- 建立分析服务
- 建立 Web 原型
- 定义统一数据结构
- 打通测试

- 可运行接口

- 接基金净值与基础信息
- 建立数据同步任务

- 增加经理与主题分析

## Phase 3：研究能力

- 公告解析
- 文章摘要
- 向量检索
- 证据链输出

## Phase 4：策略能力

- 回测
- 模拟交易
- 调仓提醒
- 风险预算与再平衡

## Phase 5：多端发布

- Web 正式版
- Windows 桌面版
- Mobile 客户端
- 登录与同步
